## Browser (claude-in-chrome)

You have access to a Chrome browser. When tasks require opening websites, navigating to URLs, reading web content, or interacting with web UIs, use the browser tools. Do not say you cannot open a browser — you can.

Common browser tasks: open a URL, take a screenshot, read page content, click buttons, fill forms.
